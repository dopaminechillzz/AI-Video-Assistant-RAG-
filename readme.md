## ✨ Features

### 🎯 Core Capabilities
- **📥 YouTube Video Processing** — Download and extract audio from any YouTube URL
- **🎙️ AI Transcription** — Powered by OpenAI Whisper for accurate speech-to-text (supports English & Hinglish)
- **📝 Smart Summarization** — Concise, professional summaries generated by Mistral AI
- **🔍 Insight Extraction** — Automatically identify action items, key decisions, and open questions
- **💬 Conversational RAG Chat** — Ask questions about video content using Retrieval-Augmented Generation
- **📄 Full Transcript Search** — Search through transcripts with keyword highlighting

### 🎨 User Interface
- **YouTube-Inspired Dark Theme** — Clean, familiar interface with `#0f0f0f` dark background
- **Search-Bar First Design** — Centered URL input with red "Analyze" button, just like YouTube
- **Tabbed Results** — Organized into Summary, Transcript, Insights, and Chat sections
- **Metric Dashboard** — At-a-glance counts of action items, decisions, and questions
- **Download Support** — Export summaries and transcripts as text files
- **Responsive Layout** — Works on desktop and tablet

### 🛡️ Privacy & Performance
- **Local Processing** — All computation runs on your machine (only API calls to Whisper and Mistral)
- **Chunked Audio Processing** — Handles long videos by splitting into manageable segments
- **Vector Database** — ChromaDB-powered semantic search for accurate Q&A

## 🚀 How It Works

```
1️⃣ INPUT                       2️⃣ PROCESS                        3️⃣ OUTPUT
┌──────────┐    ┌──────────────────────────────┐    ┌──────────────────┐
│ YouTube   │───▶│ yt-dlp → Audio Extraction   │───▶│ AI Summary       │
│ URL       │    │ → Whisper Transcription     │    │ Action Items     │
│ or        │    │ → Text Chunking             │    │ Key Decisions    │
│ File Upload│   │ → Embedding Generation      │    │ Open Questions   │
└──────────┘    │ → ChromaDB Vector Store      │    │ Chat Interface   │
                └──────────────────────────────┘    └──────────────────┘
```

### Pipeline Steps

1. **Input** — Paste a YouTube URL or upload a video/audio file (MP4, MOV, AVI, MKV, WAV, MP3)
2. **Audio Extraction** — Uses `yt-dlp` + FFmpeg to download and convert audio to 16kHz mono WAV
3. **Transcription** — OpenAI Whisper converts speech to text with high accuracy
4. **Summarization** — Mistral AI generates a concise summary and contextual title
5. **Insight Extraction** — Separate Mistral passes extract action items, key decisions, and questions
6. **Vector Embedding** — Transcript chunks are embedded using HuggingFace `all-MiniLM-L6-v2`
7. **RAG Setup** — ChromaDB stores embeddings for semantic search and Q&A
8. **Interactive Chat** — Ask natural language questions and get cited, contextual answers

| **Audio Processing** | yt-dlp, FFmpeg, Pydub | Audio download and conversion |

## 📦 Installation

### Prerequisites

- **Python 3.10 or higher**
- **FFmpeg** (required for audio processing)
  - **Windows**: Download from [ffmpeg.org](https://ffmpeg.org/download.html) and add to PATH
  - **Linux**: `sudo apt install ffmpeg`
  - **macOS**: `brew install ffmpeg`
- **Git** (for cloning the repository)

### Step-by-Step Setup

```bash
# 1. Clone the repository
git clone https://github.com/dopaminechillzz/AI-Video-Assistant-RAG-.git
cd AI-Video-Assistant-RAG-

# 2. Create a virtual environment
# Windows
python -m venv .venv
.venv\Scripts\activate

# Linux / macOS
python3 -m venv venv
source venv/bin/activate

# 3. Install dependencies
pip install -r requirements.txt

# 4. Set up environment variables
# Create a .env file in the project root:
#   MISTRAL_API_KEY=your_mistral_api_key_here
```

### Configuration

| Variable | Description | Default |
|---|---|---|
| `MISTRAL_API_KEY` | API key for Mistral AI services | **(required)** |
| `WHISPER_MODEL` | Whisper model size (`tiny`, `base`, `small`, `medium`, `large`) | `small` |

### Run the Application

```bash
streamlit run appz.py
```

Then open your browser to `http://localhost:8501`.

## 🔧 Development

### Extending the Application

| Want to... | File to modify |
|---|---|
| Add a new LLM provider | `core/summarizer.py`, `core/extractor.py`, `core/rag_engine.py` — update `get_llm()` |
| Change embedding model | `core/vector_store.py` — update `EMBEDDING_MODEL` constant |
| Adjust audio chunking | `utils/audio_processor.py` — modify chunk duration |
| Add new export formats | `appz.py` — add new download buttons |
| Customize the UI theme | `style.css` — update CSS variables and component styles |
| Add a new analysis tab | `appz.py` — add a new tab to the `st.tabs()` list |
